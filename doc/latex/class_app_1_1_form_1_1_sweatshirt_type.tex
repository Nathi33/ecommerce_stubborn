\doxysubsection{Description détaillée}
Formulaire pour la création ou modification d\textquotesingle{}un Sweatshirt. 

Ce formulaire permet de renseigner \+:
\begin{DoxyItemize}
\item le nom du sweatshirt,
\item son prix,
\item le stock disponible pour chaque taille (XS à XL),
\item une image facultative,
\item une option pour le mettre en avant. 
\end{DoxyItemize}
