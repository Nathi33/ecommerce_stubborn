\doxysubsection{Description détaillée}
Formulaire permettant à un utilisateur de changer son mot de passe. 

Ce formulaire utilise un champ de mot de passe répété pour forcer la confirmation du mot de passe. Le mot de passe doit avoir une longueur minimale de 8 caractères et ne doit pas être vide. Le champ n\textquotesingle{}est pas mappé à une propriété de l\textquotesingle{}entité (mapped\+: false), car l\textquotesingle{}encodage du mot de passe est généralement géré manuellement dans le contrôleur.
