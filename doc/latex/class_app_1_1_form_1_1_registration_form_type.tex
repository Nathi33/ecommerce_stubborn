\doxysubsection{Description détaillée}
Formulaire d\textquotesingle{}inscription d\textquotesingle{}un nouvel utilisateur. 

Ce formulaire permet de collecter les informations nécessaires à la création d\textquotesingle{}un compte utilisateur \+: email, nom d\textquotesingle{}utilisateur, adresse de livraison et mot de passe (avec confirmation). Les mots de passe ne sont pas mappés à l\textquotesingle{}entité User afin d\textquotesingle{}être encodés dans le contrôleur.
