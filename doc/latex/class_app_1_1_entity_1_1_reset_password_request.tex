\doxysubsection{Description détaillée}
Entité représentant une demande de réinitialisation de mot de passe. 

Cette entité est utilisée par le bundle Symfony\+Casts Reset\+Password pour gérer la logique de création, validation et expiration des Tokens de réinitialisation.
