◆ addToCart()

App\Controller\CartController::addToCart ( Request $request,
Sweatshirt $sweatshirt,
SessionInterface $session )

Permet d'ajouter un sweat-shirt au panier.

Cette méthode récupère les informations du sweat-shirt, la taille choisie par l'utilisateur, et les ajoute dans le panier stocké en session.

@Route("/cart/add/{id}", name="app_cart_add", methods={"POST"})

Paramètres
Request$requestLa requête HTTP contenant les informations envoyées par le formulaire.
Sweatshirt$sweatshirtL'entité sweat-shirt à ajouter au panier.
SessionInterface$sessionL'interface pour gérer les données de session.
Renvoie
Response Redirige vers la page du panier après l'ajout du sweat-shirt.

◆ removeFromCart()

App\Controller\CartController::removeFromCart ( SessionInterface $session,
int $index,
Request $request )

Permet de supprimer un sweat-shirt du panier.

Cette méthode récupère le panier depuis la session et supprime l'élément à l'index spécifié. Elle vérifie également le token CSRF pour assurer la sécurité de la suppression.

@Route("/cart/remove/{index}", name="app_cart_remove", methods={"POST"})

Paramètres
SessionInterface$sessionL'interface pour gérer les données de session.
int$indexL'index de l'article à supprimer dans le panier.
Request$requestLa requête HTTP contenant le token CSRF.
Renvoie
Response Redirige vers la page du panier après la suppression de l'article.
